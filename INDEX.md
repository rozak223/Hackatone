## 🎯 What This Project Does

**SYNCHAIN AI** is an intelligent supply chain system that:

1. **Predicts Product Demand** using Machine Learning (XGBoost)
2. **Recommends Restock Actions** based on predictions
3. **Provides Explainable AI** showing which factors affect predictions
4. **Automates Workflows** through API integration

**For UMKM (Small Businesses):** Reduces costs by 20-50% through better inventory management

## 🔧 Installation

### Option A: Automatic (Recommended)
```bash
# Copy-paste to terminal:
cd Hackatone-main && python -m venv .venv && .venv\Scripts\activate && pip install -r requirements.txt && python demo.py
```

### Option B: Manual (Step-by-step)
```bash
# 1. Create virtual environment
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run demo
python demo.py

# 4. (Optional) Setup Kaggle for real data
# See PROJECT_STRUCTURE.md > Setup Kaggle API
```
