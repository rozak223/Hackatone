🔧 DASHBOARD ELEMENTS DIJELASKAN
═════════════════════════════════════════════════════════════════════════════

HEADER SECTION:
   Total Produk: Berapa item yang di-monitor
   Butuh Restock: Berapa item yang perlu order sekarang
   Alerts: Berapa notifikasi aktif
   Online: Status sistem (hijau = berjalan)

STATUS BAR:
   Stok Saat Ini: Inventory sekarang
   Prediksi Demand: AI predict berapa butuh

PROGRESS BAR:
   Panjang bar = percentage stok vs forecast
   Penuh (hijau) = OK
   Separuh = Warning
   Kosong (merah) = Critical

PRIORITY LEVELS:
   HIGH: Order sekarang juga
   MEDIUM: Order dalam beberapa jam
   LOW: Pesan kapan saja

STATUS BADGE:
   ✅ OK: Inventory aman
   ⚠️ WARNING: Perlu diperhatikan
   🔴 ACTION NEEDED: Harus action sekarang

BUTTONS:
   Edit Stok: Update stock cepat
   Details: Lihat info lengkap
   Refresh: Manual refresh

💡 TIPS & TRICKS
═════════════════════════════════════════════════════════════════════════════

1. MONITOR DARI HP:
   • Akses http://localhost:5000 dari HP
   • Check stok di toko sambil jalan-jalan
   • Update stok dari mana saja

2. MULTIPLE WINDOWS:
   • Buka 2 browser window
   • Satu untuk lihat, satu untuk update
   • Lihat real-time changes

3. FULL SCREEN:
   • F11 untuk full screen
   • Better view di monitor toko

4. QUICK CHECKS:
   • Lihat card warna merah = action needed
   • Fokus ke alerts section dulu
   • Prioritaskan HIGH alerts

5. BATCH UPDATE:
   • Update beberapa stok sekaligus
   • Prediksi all calculated instantly
   • Lihat mana yang perlu order

❓ FAQ
═════════════════════════════════════════════════════════════════════════════

Q: Dashboard tidak load?
A: • Pastikan running: python realtime_dashboard.py
   • Buka: http://localhost:5000 (bukan https)
   • Cek firewall port 5000

Q: Data tidak update?
A: • Tunggu 3 detik (refresh interval)
   • Klik "Refresh" button
   • Check console untuk errors

Q: Prediksi tidak akurat?
A: • Demo pakai sample data
   • Lebih akurat dengan real data
   • Train dengan data Anda sendiri

Q: Mau integrate dengan sistem lain?
A: • Lihat API endpoints di API_GUIDE.md
   • Use /api/inventory/update untuk push data
   • See mobile_api.py untuk contoh

Q: Mau notifikasi via email/SMS?
A: • Ada contoh di EXAMPLES.md
   • Gunakan n8n untuk automation
   • Connect ke email/SMS service
