## 🚀 Quick Start Guide

### Option 1: Fastest Way (5 seconds)
```bash
# Demo with dummy data (no setup needed)
python demo.py
```

### Option 2: Full System with Real Data (60 seconds)
```bash
# 1. Setup Python environment
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac

# 2. Install dependencies
pip install -r requirements.txt

# 3. Setup Kaggle API (see section below)

# 4. Run main system
python synchain_ai.py
```

### Option 3: Production Setup (Advanced)
```bash
# 1-2. Same as Option 2

# 3. Use advanced version
python synchain_advanced.py

# 4. Use in your application
from synchain_advanced import SynChainAI_Advanced
ai = SynChainAI_Advanced()
ai.load_model()
prediction = ai.get_predictions_with_confidence({'product_id': 5})
```

## 🔑 Setup Kaggle API

### Step 1: Get API Key
1. Go to https://www.kaggle.com/settings/account
2. Click "Create New API Token"
3. File `kaggle.json` will download

### Step 2: Place API Key
**Windows:**
```powershell
mkdir $env:USERPROFILE\.kaggle
Copy-Item kaggle.json $env:USERPROFILE\.kaggle\
```

**Linux/Mac:**
```bash
mkdir ~/.kaggle
cp kaggle.json ~/.kaggle/
chmod 600 ~/.kaggle/kaggle.json
```

### Step 3: Verify
```bash
python -c "import kagglehub; print('✅ Kaggle API ready')"
```

## 🚨 Common Issues & Solutions

### Issue: "Module not found: kagglehub"
```bash
pip install kagglehub
```

### Issue: "Kaggle API error"
```bash
# Check API key location
ls ~/.kaggle/kaggle.json  # Linux/Mac
dir $env:USERPROFILE\.kaggle\  # Windows
```

### Issue: "Model not found: synchain_model.pkl"
```bash
# Run training first
python synchain_advanced.py
```

### Issue: Memory error with large datasets
```python
# Process in chunks
chunk_size = 50000
# See API_GUIDE.md for details
```
